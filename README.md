# M4M (Music for Millennials)

밀레니얼 세대를 위한 세대공감 음악 추천 및 검색 서비스 M4M<br>
밀레니얼 세대들이 추억여행을 떠날 수 있는 1992년 부터 2009년까지 노래들을 준비했습니다!<br>
간단한 질문들을 통해 여러분의 평소 성향에 어울릴만한 노래를 추천해드립니다!<br>
추천받은 노래들을 친구/동료들과 공유해보면서 추억여행을 떠나보세요!
